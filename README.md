# Meetup Clone

## Live Demo
https://meetup-clone-three.vercel.app/

## Stack
- Angular 21 (Signals, Control Flow)
- TailwindCSS
- PrimeNG

## Features
- Home with event listing
- Event detail page
- Authentication (mock JWT)
- Protected routes with guards
- Responsive design (mobile first)

## Architecture
- Feature-based structure
- Shared reusable components
- Centralized event service (mock data)

## Auth
Simulated authentication using localStorage
